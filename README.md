## 📥 安装说明

### macOS (Apple Silicon)
1. 下载 `AlphaBot-macOS-arm64.tar.gz`
2. 解压: `tar -xzf AlphaBot-macOS-arm64.tar.gz`
3. 将 `AlphaBot.app` 拖到应用程序文件夹
4. 首次运行右键点击，选择「打开」

### macOS (Intel)
1. 下载 `AlphaBot-macOS-x64.tar.gz`
2. 解压: `tar -xzf AlphaBot-macOS-x64.tar.gz`
3. 将 `AlphaBot.app` 拖到应用程序文件夹
4. 首次运行右键点击，选择「打开」

### Windows
1. 下载并解压 `AlphaBot-Windows-x64.zip`
2. 运行 `AlphaBot.exe`
